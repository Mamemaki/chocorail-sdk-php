# Chocorail SDK for PHP

このリポジトリは[ちょこれ〜る](https://chocorail.com/)のPHP用SDKです。[ちょこれ〜ちょこれ〜るAPI](https://chocorail.com/references/chocorail-api/)の呼び出し用メソッドやWebアプリケーション構築用のヘルパーメソッドが含まれます。

## インストール
PHP SDKのインストールはcomposerを使うことを推奨します。以下のコマンドでインストールすることができます。

```shell
composer require mamemaki/chocorail-sdk-php
```

## 使い方

[ちょこれ〜ちょこれ〜るAPI リファレンス](https://chocorail.com/references/chocorail-api/)を参照ください。
また、<a href="https://github.com/Mamemaki/chocorail-spapp-skeleton" target="_blank">スケルトンプロジェクト</a>も参考にしてください。
